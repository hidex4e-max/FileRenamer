# ふぁいるりねーまー改

アニメタイトルのフォルダをドラッグ＆ドロップするだけで、しょぼいカレンダーからタイトルを自動取得してファイル名を一括リネームするツールです。

## 機能

- **フォルダD&D**: アニメタイトルのフォルダをドロップすると再帰的に動画ファイルを収集
- **自動タイトル抽出**: フォルダ名からタイトルを自動認識
- **しょぼかる連携**: TID検索・タイトル取得・サブタイトル取得
- **自動TID取得**: D&D時にフォルダ名で自動検索
- **Undo**: リネームを元に戻せる
- **直接リネーム**: F2キーでファイル名を直接編集
- **一括削除**: Delキーでリストから削除、Ctrl+Aで全選択
- **ブラウザ機能**: メニューからしょぼかるブラウザを開いてTIDを取得

## 使い方

1. 動画ファイルが入ったフォルダをアプリにドラッグ＆ドロップ
2. タイトルが自動検索されるのを確認
3. 話数・タイトルを調整
4. 「確認」でプレビュー
5. 「リネーム」で実行

## 必要環境

- .NET Framework 4.6.2
- Windows 7 / 8 / 10 / 11
